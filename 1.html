HTML

Hypertext Martkup Language(HTML) is a markup language for creating a creating web pages. Webpages are usually viewed in a web browser. They can include writing, links, pictures, and even sound and video. HTML is used to mark and describe each of these kinds of content so the web browser can display them correctly. HTML can also be used to add meta information to a webpage. Meta information is usually not shown by web browsers and is data about the web page is used to manipulate HTML elements and CSS styles.
